💡 Tips

  • Ctrl+Enter: Run all checks
  • Green ✅: File/endpoint working
  • Red ❌: File/endpoint not accessible
  • Orange ⚠️: Possible configuration issue

If all checks fail: Files may not be uploaded or domain not configured properly.

If some work: Check file permissions and .htaccess configuration.
