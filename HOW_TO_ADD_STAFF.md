# How to Add Staff Members to "Meet Our Family"

## 📍 Where to Find It

Admin Dashboard → **"👥 Meet Our Family"** tab

---

## 📋 Form Fields Explained

### 1. **Name *** (Required)
- Enter the full name of the staff member
- Example: `John Doe`

### 2. **Position *** (Required)
- Enter the person's role/position
- Examples:
  - `OIC – National General Secretary`
  - `Secretary for Finance`
  - `National Program Secretary`
  - `Director of Operations`

### 3. **Department/Group** (Optional)
Choose which department/category they belong to. Options:
- National General Secretary
- National Program Secretary
- Secretary for Finance
- Secretary for Member Association
- Secretary for Operation
- Other

This groups staff members by department on the "Meet Our Family" page.

### 4. **Display Order** (Optional)
- Determines the order staff appear within their department
- Use `0, 1, 2, 3...` for first, second, third, etc.
- Staff with lower numbers appear first
- Default is `0`

### 5. **Upload Photo** OR **Photo URL**
Choose one method to provide a photo:

#### Option A: Upload Photo
- Click "Choose File" button
- Select an image from your computer (JPG, PNG, WebP, etc.)
- Recommended size: 400×500px or larger
- File size: up to 5MB

#### Option B: Use Photo URL
- If you have a URL to the person's photo, paste it here
- Example: `https://example.com/images/john-doe.jpg`
- This is useful if photo already hosted elsewhere

## 🚀 Step-by-Step Guide

### **Adding a New Staff Member**

1. **Scroll to the form** at the top of the page
2. **Fill in Name**
   ```
   Name: Rev. Maria Santos
   ```

3. **Fill in Position**
   ```
   Position: OIC – National General Secretary
   ```

4. **Select Department** (optional)
   ```
   Department: National General Secretary
   ```

5. **Set Display Order** (optional)
   ```
   Display Order: 1  (for first in this department)
   ```

6. **Add Photo** - Choose ONE method:
   - **Upload**: Click "Choose File" and select photo from computer
   - **OR URL**: Paste photo web URL

7. **Click "Add Staff"** button
   - Form will submit
   - Should show ✅ "Staff added successfully"

8. **Verify** 
   - Staff member now appears in the "Staff Directory by Department" section below the form
   - Check they appear correctly on the public "Meet Our Family" page

## ✏️ Editing Existing Staff

1. **Scroll down** to "Staff Directory by Department"
2. **Find the person** you want to edit
3. **Click "Edit"** button on their card
4. **Make changes** to any fields
5. **Click "Update Staff"** button
6. **Verify** changes appear on public site

---

## 🗑️ Deleting a Staff Member

1. **Find the person** in staff directory
2. **Click "Delete"** button on their card
3. **Confirm** deletion
4. **Person removed** from all locations

## 🎨 Best Practices

### Photo Guidelines
- **Aspect Ratio**: Square (1:1) works best
- **Size**: At least 400×400px
- **Quality**: Clear, well-lit photos
- **Format**: JPG or PNG
- **File Size**: Under 5MB

### Ordering Example
```
Department: National General Secretary
├── Order 0 → Rev. Emanuel Reyes (displays first)
├── Order 1 → John Smith (displays second)  
└── Order 2 → Maria Santos (displays third)

Department: Secretary for Finance
├── Order 0 → Jane Doe (displays first)
└── Order 1 → Robert Wilson (displays second)
```

### Position Title Tips
- Use official titles for consistency
- Include "OIC" (Officer-in-Charge) if applicable
- Format: `Title – Department` where applicable
- Examples:
  - ✅ `OIC – National General Secretary`
  - ✅ `Secretary for Finance`
  - ✅ `Chairman of the Board`
  - ❌ `General Secretary (use full title)`

## ⚠️ Troubleshooting

### Photo not showing?
- Check file was uploaded successfully
- Check image URL is correct and accessible
- Try uploading a smaller image
- Verify image format (JPG, PNG, WebP)

### Staff not appearing on public site?
- Reload/refresh the page (Ctrl+F5)
- Check staff was saved (look for ✅ success message)
- Verify staff appears in directory section below form
- Check Display Order is not hidden

### Position not displaying correctly?
- Make sure you entered text in the Position field
- Check for typos
- Keep titles reasonable length

## 💡 Tips & Tricks

### Bulk Reordering
If you need to reorder staff:
1. Edit each person
2. Change their "Display Order" number
3. Click "Update Staff"
4. Repeat for other staff

### Department Management
- Unassigned staff go to "Other" department
- To move between departments, just edit and change Department dropdown
- Can create custom groups - just type in the department field

### Photo Optimization  
For best quality:
1. Use high-res photos (600×600px or larger)
2. Crop to square
3. Use modern format (WebP if possible)
4. Compress before uploading to save server space
